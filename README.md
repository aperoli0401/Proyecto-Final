# Proyecto-Final

## Idea del proyecto 

Esta aplicación consiste en un gestor de gastos personales desarrollado en Java con interfaz gráfica. Su objetivo es permitir a los usuarios registrar, organizar y analizar sus ingresos y gastos de forma sencilla e intuitiva.

El sistema permitirá añadir transacciones, clasificarlas por categorías y consultar el historial financiero, ofreciendo una visión clara del estado económico del usuario.


## Necesidad del proyecto

Muchas personas tienen dificultades para controlar en qué gastan su dinero día a día. Esto puede provocar desorganización financiera y dificultad para ahorrar.

Esta aplicación busca cubrir esa necesidad proporcionando una herramienta que:

    1. Permite registrar todos los movimientos económicos (ingresos y gastos)

    2. Facilita la organización mediante categorías

    3. Ayuda a visualizar el balance total disponible

    4. Permite analizar hábitos de consumo

De esta forma, el usuario podrá tomar decisiones financieras más informadas y mejorar su control económico personal.

## Usuarios

La aplicación está dirigida a cualquier persona que quiera gestionar sus finanzas personales, especialmente:

    1. Estudiantes que desean controlar sus gastos mensuales

    2. Trabajadores que buscan organizar sus ingresos y gastos

    3. Usuarios sin conocimientos técnicos que necesiten una herramienta sencilla

En versiones más avanzadas, se podría incluir soporte para múltiples usuarios con cuentas independientes.
